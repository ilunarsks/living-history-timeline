About This Project

This interactive timeline is the result of a semester‑long, project‑based learning unit. Students work together using GitHub to research, write, code, and review content about the Space Race (1955–1975).

The project simulates a real‑world development team: tasks are tracked as Issues, contributions are made via Pull Requests, and quality is ensured through peer code review.

Learning objectives include collaborative development, project management, content curation, web development, and quality assurance.

View the GitHub Repository

How to Contribute

  1. Fork the repository to your own GitHub account.
  2. Clone your fork locally and create a new branch for your task.
  3. Claim an Issue from the Issues tab and assign it to yourself.
  4. Research the event and add it to data/events.json.
  5. Commit & push your changes, then open a Pull Request.
  6. Link the Issue in the PR description with Closes #XX.
  7. Address feedback from peer reviewers before merging.

Each merged Pull Request automatically adds a new event to the timeline!
